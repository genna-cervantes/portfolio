# Genna Cervantes Portfolio

This is my personal portfolio.

It mainly exists so I have a place to showcase my notes and writing about
engineering problems, patterns, and decisions I run into while building software.

The site was pretty much vibe coded with Claude Code and Codex, then adjusted as
needed for my own content and preferences.

## Tech

- Next.js
- React
- TypeScript
- Markdown notes rendered with `react-markdown`

## Run Locally

```bash
npm install
npm run dev
```

Open http://localhost:3000.
